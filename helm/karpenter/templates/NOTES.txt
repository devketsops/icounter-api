Karpenter has been deployed successfully!

To verify the Karpenter controller is running:
  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name=karpenter

To check NodePool status:
  kubectl get nodepools

To check EC2NodeClass status:
  kubectl get ec2nodeclasses

To view Karpenter logs:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name=karpenter -f

To test scaling, create a deployment that requests resources:
  kubectl scale deployment <your-deployment> --replicas=5
  kubectl get nodes -w
